About tau-fst

tau-FST is a library for constructing and combine a type of fuzzy logic-based finite-state transducers named TFFST. Finie-state transducers are automata where each transition has an input label, an output label, and, possibly, a wight. In the case of TFFSTs, those labels are some fuzzy functions named Tautness Functions. In the context of network management, TFFSTs are both, a mathematical model and a reasoning-machine implementation model that made possible to bring declarative programming and autonomous behavior to network devices, and in particular to resource-constrained devices. The existent core of TFFST operations has a long waiting list of extensions needed to make it more usable, from a module to translate high-level operation policies to low level finite state transducers, to a more efficient heuristic to minimize fuzzy-logic expressions.
